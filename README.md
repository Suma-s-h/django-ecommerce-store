# 🛒 Django E-Commerce Store

A full-stack e-commerce web application built using Django. This project enables users to browse products, manage shopping carts, place orders, and securely authenticate their accounts through a clean and responsive interface.

---

## ✨ Features

- 👤 User Registration & Login
- 🛍️ Product Catalog
- 🔍 Product Details
- 🛒 Shopping Cart
- 📦 Order Management
- 💳 Checkout Workflow
- 🖼️ Product Image Support
- 🔐 User Authentication
- 📱 Responsive Design
- ⚙️ Django Admin Panel

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Suma-s-h/django-ecommerce-store.git
```

### Navigate to the project

```bash
cd django-ecommerce-store
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Apply migrations

```bash
python manage.py migrate
```

### Run the development server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
